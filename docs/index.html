Acerca de nosotros

Un equipo que entiende el inventario real

SIPU nace para resolver el desorden que viven las PYMES con datos incompletos. Somos estudiantes de Tecnologia en Desarrollo de Software de la Universidad del Valle, sede Zarzal.

Mision

Crear un sistema de inventario simple, confiable y escalable que reduzca errores y mejore la rentabilidad de los negocios locales.

Vision

Ser la herramienta preferida para la gestion de inventarios en PYMES colombianas.
